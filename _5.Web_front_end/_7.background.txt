背景相关属性
背景颜色
    属性:
        background-color
    取值:
        颜色值
背景图片
    属性
        background-image
    取值
        url('')
    相关属性
        背景图片重复平铺显示 background-repeat
        取值
            repeat(默认值)
                当图片尺寸小于元素尺寸,会自动沿着水平和垂直方向重复平铺
                直到铺满元素
            repeat-x
                设置图片沿水平方向平铺
            repeat-y
                设置图片沿垂直方向平铺
            no-repeat
                设置图片不平铺
        背景图片尺寸
            background-size
            取值
                像素值
                    取两个值,分别表示背景图片的宽和高;
                    取一个值,设置背景图片宽度,高度等比缩放
                百分比
                    百分比取一个值或者两个值等同于像素的取值情况
                cover
                    覆盖,等比拉升至足够大完全覆盖元素
                contain
                    包含,等比拉伸图片至刚好被元素容纳的最大尺寸
        背景图片位置
            background-position
            取值
                像素值 x y
                    x表示背景图片水平偏移距离,正值表示向右
                    y表示背景图片垂直偏移距离,正值表示向下
                    默认背景图片在元素左上角显示
                百分比
                    0% 0%       左上角显示
                    50% 50%     居中显示
                    100% 100%   右下角显示
                方位值
                    水平方向    left center  right
                    垂直方向    top center  bottom
                    设置方位值  如果缺少某个方向,默认为center
                使用场景
                    精灵图技术,网页开发过程为了节省资源,减少网页请求,通常会用一组小图
                    以一张图片的方式存储通过一次网络请求加载图片,通过background
                    控制图标切换显示
        背景属性简写
            属性
                background
            取值
                color  url()   repeat position
            示例    
                background: pink  url(path)  no-peat 10px 10px
            注意
                background-size 单独设置
文字和字体相关属性
    字体大小
        font-size
        取值 像素值
    指定字体名称
        font-family 字体名称
        语法注意
            字体名称如果是中文,或者由多个英文单词组成,必须加引号
            可以设置多个字体名称,做备用字体,名称之间用逗号隔开
    设置字体加粗
        属性
            font-weight
        取值
            可以取关键字
                bold(加粗显示)/normal(默认,正常显示)
            取无单位的整百数
            400正常
            700等同bold
    设置字体样式
        属性
            font-style
        取值
            normal  默认正常显示
            italic  斜体显示
            oblique 文本倾斜显示
                一般作为italic的替换样式,可以实现斜体效果
                在某些情况下,可以指定倾斜角度
    字体属性简写
        属性    font
        取值    style   weight    size    family(顺序强制)
        语法注意
            size,family为必填项
    
文本相关
    文本颜色
        属性:color
        取值:颜色值
    文本水平对齐方式
        属性
            text-align
        取值
            left(默认)/center/right
    文本装饰线
        属性
            text-decoration
        取值
            underline       下划线
            overline        上划线
            line-through    删除线
            none            取消装饰线
    行高
        属性
            line-height
        取值
            像素值
        注意
            所有文本在其所属行中都是垂直居中的
        使用场景
            行高可以设置一行的垂直居中
                行高和元素的高度保持一致
            行高可以实现文本在元素上下位置的微调
            示例
                1.{
                    height:100px;
                    line-height:120px;
                }
                2.{
                    height:100px;
                    line-height:70px;
                }
