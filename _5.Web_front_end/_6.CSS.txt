CSS
简单介绍
    css: Cascading Style Sheet 层叠样式表
    作用:调整页面元素的显示外观,实现网页布局
使用
    在HTML文档中使用CSS样式,有三种方式
    1.内联样式/行内样式
        特点:在标签中通过style属性,为元素设置样式
        语法
            <标签名 style='属性:值;属性2:值;'></标签名>
        常用的CSS属性
            1)font-size
                设置元素的字体大小,取像素值,默认为16像素
            2)color
                用来设置元素的字体颜色,取颜色值
            3)background-color
                设置元素的背景颜色,取颜色值
    2.文档内嵌
        特点:在文档中使用<style></style>标签,为文档元素设置样式 
        语法
            <style type='text/css'>
                选择器{属性:值
                      属性2:值
                      ...
                     } 
                选择器{属性:值
                      属性2:值
                      ...
                     }                 
            </style>
            选择器:用来匹配文档中的元素并为其设置样式
            et:
                p{
                    color:red;
                    font-size:32px
                }
                通过标签名匹配文档中所有的元素,叫标签选择器
    3.外链
        特点:在HTML文档中那个引入外部的样式表文件
        在HTML文档中使用
            <link rel='stylesheet' href='xx.css' type='text/css'>
        然后在外链部署css样式
css特征
    层叠性
        可以为一个元素设置多个样式,共同起作用
            p{
                color:red;
                background-color:green;
                fontsize:32px;
            }       
    继承性
        子元素可以继承父元素或者祖先元素的某些css样式
        例如,大部分的文本属性都可以被继承
            块元素默认宽度与父元素保持一致  h1 p div
    优先级
        从低到高依次为:
        浏览器默认样式
        文档内嵌样式/外链文件中的样式,以代码书写顺序为准(以最后的样式为准,包括外链文件)
        行内样式(最高)
        如果发生冲突,参考优先级,决定元素最终样式
选择器
    选择器作用
        根据不同的选择器,匹配文档的相应元素,并为其设置样式
    选择器分类
        标签选择器
            作用
                作用根据标签名匹配文档中所有的该元素
            语法:
                标签名{
                    属性:值
                }
            练习
                1,创建超链接标签
                2.设置文本颜色为红色
                3.取消下划线 text-decoration : none;
                示例:08_
        id选择器
            注意
                所有的元素具有id属性,属性值自定义
                id属性具有唯一性
            语法
                #id属性值{
                    样式
                }
            示例
                *单位不可省略
                #box{
                    width:200px;
                    height:200px;
                    background-color:red;
                }
                <div id='box'></div>
        class选择器
            根据标签的class属性匹配元素,可以复用
                .class属性值{
                    样式
                }
            特殊用法
                1.
                2.class属性值可以出现多个使用空格隔开
                    et 
                    class='c1 c2 c3'
        群组选择器
            可以为一组元素,设置共同样式
            标签选择器和类选择器结合使用
            语法
                选择器1,选择器2,选择器3{
                    样式
                }
            常见于清楚浏览器默认样式
        后代选择器
            特点
                匹配满足要求的所有后代元素
            语法
                选择器1 选择器2{
                    样式
                }
            选择器1 表示匹配父元素
            选择器2 表示匹配后代元素
