CSS
简单介绍
    css: Cascading Style Sheet 层叠样式表
    作用:调整页面元素的显示外观,实现网页布局
使用
    在HTML文档中使用CSS样式,有三种方式
    1.内联样式/行内样式
        特点:在标签中通过style属性,为元素设置样式
        语法
            <标签名 style='属性:值;属性2:值;'></标签名>
        常用的CSS属性
            1)font-size
                设置元素的字体大小,取像素值,默认为16像素
            2)color
                用来设置元素的字体颜色,取颜色值
            3)background-color
                设置元素的背景颜色,取颜色值
    2.文档内嵌
        特点:在文档中使用<style></style>标签,为文档元素设置样式 
        语法
            <style type='text/css'>
                选择器{属性:值
                      属性2:值
                      ...
                     } 
                选择器{属性:值
                      属性2:值
                      ...
                     }                 
            </style>
            选择器:用来匹配文档中的元素并为其设置样式
            et:
                p{
                    color:red;
                    font-size:32px
                }
                通过标签名匹配文档中所有的元素,叫标签选择器
    3.外链
        特点:在HTML文档中那个引入外部的样式表文件
        在HTML文档中使用
            <link rel='stylesheet' href='xx.css' type='text/css'>
        然后在外链部署css样式
css特征
    层叠性
        可以为一个元素设置多个样式,共同起作用
            p{
                color:red;
                background-color:green;
                fontsize:32px;
            }       
    继承性
        子元素可以继承父元素或者祖先元素的某些css样式
        例如,大部分的文本属性都可以被继承
            块元素默认宽度与父元素保持一致  h1 p div
    优先级
        从低到高依次为:
        浏览器默认样式
        文档内嵌样式/外链文件中的样式,以代码书写顺序为准(以最后的样式为准,包括外链文件)
        行内样式(最高)
        如果发生冲突,参考优先级,决定元素最终样式
选择器
    选择器作用
        根据不同的选择器,匹配文档的相应元素,并为其设置样式
    选择器分类
        标签选择器
            作用
                作用根据标签名匹配文档中所有的该元素
            语法:
                标签名{
                    属性:值
                }
            练习
                1,创建超链接标签
                2.设置文本颜色为红色
                3.取消下划线 text-decoration : none;
                示例:08_
        id选择器
            注意
                所有的元素具有id属性,属性值自定义
                id属性具有唯一性
            语法
                #id属性值{
                    样式
                }
            示例
                *单位不可省略
                #box{
                    width:200px;
                    height:200px;
                    background-color:red;
                }
                <div id='box'></div>
            示例
                08
        class选择器
            根据标签的class属性匹配元素,可以复用
                .class属性值{
                    样式
                }
            特殊用法
                1.标签选择器和类选择器结合使用
                    p.c1{
                        background-color:pink;
                    }
                    标签名必须放在前面
                2.class属性值可以出现多个使用空格隔开
                    et 
                    class='c1 c2 c3'
            示例
                10
        群组选择器
            可以为一组元素,设置共同样式

            语法
                选择器1,选择器2,选择器3{
                    样式
                }
                常见于清楚浏览器默认样式
            示例
                09
        后代选择器
            特点
                匹配满足要求的所有后代元素
            语法
                选择器1 选择器2{
                    样式
                }
                选择器1 表示匹配父元素
                选择器2 表示匹配后代元素
            示例
                10
        子选择器
            特点
                只匹配父元素的直接子元素
            语法
                选择器1>选择器2{
                    样式
                }
            示例
                10
        伪类选择器
            作用
                针对元素的不同状态,设置相应样式
            分类
                超链接伪类选择器
                    针对超链接不同状态设置样式
                动态伪类选择器
            使用
                超链接伪类选择器
                    访问前:a:link{
                        设置超链接访问之前的样式
                    }
                    访问后:a:visited{
                        设置超链接访问之后的样式
                    }
        动态伪类选择器
            1.hover
                鼠标滑过元素的状态
            2.active
                鼠标点按元素的状态,激活
                注意:
                    超链接可以设置四种样式,书写时必须按照以下顺序定义
                    遵守爱恨原则 LOVE/HATE
                        :link
                        :visited
                        :hover
                        :active
                    网页中一般会直接对a标签设置默认样式,配合:hover改变超链接文本颜色或者背景色
            3.foucs
                表示文本框或者密码框在获取焦点时的状态
                焦点状态指的是:正接受输入或编辑时状态
                示例
                    input:focus{
                        outline: none    #去掉选中蓝色边框栏
                    }

    选择器的优先级
        定义
            选择器的优先级看权重(值),权值越大,优先级越高
        基础选择器的权值
            标签选择器  1
            类选择器/伪类选择器    10
            id选择器    100
            行内样式    1000(行内样式永远最高)
        组合选择器的权重
            组合选择器,除了群组选择器,其他的选择器权值由各选择器的权值相加得到
            div span{
                color:red
            }
    尺寸和颜色单位
        尺寸
            属性
                width height
            单位
                px  默认单位,表示像素
                %   百分比单位,参照父元素对应属性的值获取尺寸
                cm  厘米
                mm  毫米
                pt  磅
                in  英寸inch 1英寸 = 2.54cm
                em  1em=16px
                rem 与字体大小相关
            颜色取值
                1.英文单词表示颜色
                2.rgb() 用三原色红绿蓝表示,美中取值范围为0~255;
                    red   rgb(255,0,0)
                    green rgb(0,255,0)
                    blue  rgb(0,0,255)
                    black rgb(0,0,0)
                    white rgb(255,255,255)
                3.rgba(r,g,b,a)
                    a表示alpha,透明,取值0-1
                    可用小数,0表示透明,1表示不透明 
                4.十六进制表示颜色
                    语法
                        十六进制取值范围   0-9,a-f
                    颜色
                        以#开头,每两位一组,代表一种三原色
                        每两位为一组与rgb()进行转换
                    示例
                        rgb(255,0,0)   #ff0000
                    简化:短16进制
                        由三位数组成,每一位表示一种三原色,浏览器会自动补充为6位16进制
                        #f00  ->  #ff0000               
HTML元素分类及特点
    块级元素
        独占一行,不与其他元素共行显示
        可以手动设置宽高
        默认宽度和父元素保持一致(table除外)
        常见块元素
            body 
            h1~h6 标题
            p
            div   
            table 列表
            form  表单
            ul 
            ol 列表
            li 列表项
    行内元素
        可以与其他元素共行显示
        默认尺寸由内容多少决定,不能手动设置宽高
        常见行内元素
            span label i b s u strong sub sup a 
    行内块元素
        可以与其他元素共行显示
        可以手动设置宽高
        常见行内块元素
            img 表单控件
    示例
        11
html标签嵌套
    块元素中可以嵌套任何类型的元素
        特殊情况
            段落标签中只能放行内元素
    行内元素只能嵌套行内<块>元素

        overflow

        hidden  溢出内容隐藏
        scroll  为元素添加水平和垂直方向的滚动条,不管内容有没有溢出
        auto    自动在溢出方向添加可用的滚动条

边框
    	CSS中认为所有的元素都是矩形区域
    	边框都是围绕元素内容出现的线条样式
	属性
	border边框
		取值
			width   像素值,表边宽宽度
			style   边框样式 可取值
						solid   实线边框
						dashed  虚线边框
						dotted  点线边框
						double  双线边框
			color   颜色值,默认为黑色
			示例
				14
		特殊用法
			边框默认具备3px的宽度,黑色的边框颜色,边框样式必须指定
			表单按钮,可以取消边框
				border:none 
	单边框设置
		单独设置元素上右下左四个方向的边框
		属性
			border-top:上边框
			border-right:右边框
			border-bottom:下边框
			border-left:左边框
		取值
			width style  color
	网页三角标的制作
		元素的width和height设置为0
		涉及三角标,必须由边框拼接而成
		设置四个方向的边框,宽度保持一致
		通过颜色控制三角标的指示
		*透明色: transparent
	圆角边框的实现
		属性
			border-radius
		取值
			取一个值
				px/%
				*最大值50%为椭圆/正圆,依据当前尺寸计算,圆角的大小依据取值大小而设定
			取两个值
				border-radius:30px  50px;
				第一个值为上下的圆角程度
				第二个为左右的圆角程度
				可以超过50%(只能单独给值才行)
			取三个值
				border -radius:10px 20px 30px
				第一个值 上
				第二个值  左右
				第三个值  下
			取四个值
				分别代表上 右 下 左
				四个角不能同时大于50%
		示例
			15
	盒阴影
		为元素添加阴影效果
		属性  
			box-shadow
		取值  
			offset  阴影的水平偏移距离
			offset-y 阴影的垂直偏移距离,取像素值
			blur	阴影的模糊程度
			spread	阴影的延伸距离,取像素值
			color	阴影颜色,默认为黑色
		浏览器的坐标系
			浏览器和元素本身都包含坐标系
			默认以左上角为原点(0,0),向左,向下都分别是x,y轴的正方向
盒模型	
	介绍
		css中,认为所有元素皆为框
			元素在文档中的最终占据尺寸是由:内容尺寸,内边距大小,边框,外边距大小决定的
		计算元素的最终占据大小
			在标准盒模型下
				最终宽度为 =width +左右内边距padding + 左右边框border + 左右外边距margin
				最终高度为 width + 上下内边距 + 上下边框 + 上下外边距
			其他盒模型元素尺寸的计算(表单元素)
				元素设置的width,height表示包含内容,内边距,边框在内的总宽度和总高度
				最终宽度 = width+左右外边距
				最终高度 = height+上下外边距
	外边距
		定义
			元素和元素之间的距离
		简写属性
			margin 表示外边距
			取值
				像素值px(常用)
				百分比%
			取值情况
				margin:10px 上右下左四个方向添加10px外边距
				margin:10px 20px 
					上下边距为10px 左右外边距为20px
				margin 10px 20px 30px
					上10,左右20,下30
