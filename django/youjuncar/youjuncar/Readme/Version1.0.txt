优骏二手车展示网站v1.0功能规划(极简版)

    第一版仅限于汽车展示,以及调用电话api,和高德跳转的api,尽可能简化,
    不包含登录注册功能,预付订金功能,推荐功能,汽车信息登记功能,等待

    功能1:
        制作前台页面:展示已经有的车辆,包括的车的各类详细信息(参考瓜子二手车)
        

    功能2
        制作后台页面,需要登陆管理员账户,上传汽车信息,加入数据库,根据汽车,
        售出车辆则从数据库逻辑删除(isActive=False)
    
    功能3(待定)
        展示收车(填写信息,直接联系店长)

    功能4:
        汽车的搜索功能(品牌,价格,上牌地)
    

    数据库
        (AdminUser)管理员账户     字段:账户,密码
        (Carinfo)车辆信息       字段:...(瓜子:稍等)
        字段:
            id
            品牌型号(品牌加型号)
            车型(紧凑型,宽敞型等)
            颜色
            上牌时间
            表显里程
            上牌地
            排放标准
            排量
            车主报价
            车辆保障(包括是否是盗抢查封等车型,有什么保障措施等等)
            汽车详情
    
    静态文件:
        汽车的图片信息
